# ChefMate App

ChefMate là ứng dụng mobile hỗ trợ người dùng:
- Tìm kiếm công thức nấu ăn
- Gợi ý món ăn từ nguyên liệu có sẵn
- Lưu món yêu thích
- Lập kế hoạch bữa ăn

Project được phát triển bằng **React Native (Expo)** theo workflow teamwork sử dụng Git Branch & Pull Request trên GitHub.

---
